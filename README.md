# Healthcare Data Analysis Dashboard – Power BI
## Project Overview
The Healthcare Data Analysis Dashboard is an interactive Power BI visualization project that analyzes hospital patient data to uncover insights related to patient visits, satisfaction levels, waiting times, demographics, and admissions.
The dashboard provides a centralized view of hospital performance using key metrics and visual analytics. It helps healthcare administrators understand patterns in patient behavior and improve operational efficiency by using data-driven insights.
The dashboard uses interactive filters and multiple visualizations to explore healthcare data across different time periods, patient demographics, and satisfaction levels 

## Project Objective
The main objectives of this project are:
To analyze patient visit trends across different months and years
To evaluate patient satisfaction levels
To measure hospital waiting time performance
To understand patient demographic distribution
To analyze hospital admission patterns
To provide data-driven insights for healthcare management

## Questions KPIS
The dashboard aims to answer several important healthcare questions:
How many patients visited the hospital during the given period?
How does patient satisfaction vary across different scores?
What months experience the highest number of hospital visits?
How do hospital visits compare between 2019 and 2020?
What is the distribution of patients across different races?
Do waiting times vary across different patient demographics?
Are hospital visits more common in AM or PM hours?
How do hospital admissions differ by gender?

## Project Insights
Several important insights were observed:
Hospital visits slightly increased from 2019 to 2020.
Patient visits are almost evenly distributed between AM and PM hours.
Some months show higher patient traffic, indicating seasonal patterns.
The White patient group represents the largest demographic segment in the dataset.
Waiting times differ among patient groups.
Patient satisfaction scores help evaluate hospital service quality.
